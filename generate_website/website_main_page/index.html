SMALL-SCALE MAGNETIC FLUX ROPE DATABASE

This is a small-scale magnetic flux rope database. The flux rope events in this database are detected by an automated program based on Grad-Shafranov reconstruction algorithm. The data used in detecting algorithm is from Wind spacecraft. The maximum detection window is 361 minutes. The minimum duration threshold is 9 minutes.
